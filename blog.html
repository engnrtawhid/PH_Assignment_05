Question-1: What are the different ways to select an element in the DOM?

we can select a DOM element by getElementById, getElementsByClassName, getElementsByTagName, querySelector, or querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

  • innerHTML: it reads both the HTML markup and the text content of the element.
  • innerText: it returns the text as it appears on screen. It ignores HTML tags.
  • textContent: it returns only the text. Whiles innerText reads text as it is rendered on screen.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a parent element handles events for its child elements using a single event listener. This works because events bubble up from child to parent, allowing dynamic elements to be handled efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the target element and moves up to its parent elements. This means if you click a button inside a div, the event first affects the button, then the div, and then higher elements.

Question-5: How do you create, add, and remove elements using JavaScript?

You create an element using a JavaScript function, then add it to a parent element, and remove it by calling a remove function. This helps in dynamically changing the webpage content.
